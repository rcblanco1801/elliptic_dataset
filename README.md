# Elliptic Dataset
En este proyecto, se intenta realizar una implementación de aprendizaje federado distribuido con Spark y PyTorch, usando una aproximación local con arquitecturas de Redes Convolucionales de Grafos,
en el contexto del problema transductivo que se propone en el dataset https://www.kaggle.com/datasets/ellipticco/elliptic-data-set.
